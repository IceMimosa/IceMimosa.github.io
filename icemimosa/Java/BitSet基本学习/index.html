Java BitSet基本学习

1. 基本思想

Java的BitSet即BitMap。Java中，对于一个整型数字int a = 1，则 a占用4个字节，即32位。对于某些场景而言，这属于巨大的浪费，因为我们可以用 32bit来表示 0~31个十进制数字，而这就是Bitmap的基本思想。

Bitmap可以用来处理大量（海量）数据的排序，查询以及去重。且在做如用户群的交集和并集运算时，也有极大的便利。
