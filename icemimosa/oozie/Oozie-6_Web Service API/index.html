1. з®Җд»Ӣ

The Oozie Web Services API is a HTTP REST JSON API.
All responses are in UTF-8 .

Assuming Oozie is runing at OOZIE_URL ( еҰӮhttp://localhost:11000/oozie ) , the following web services end points are supported:

  • /versions
  • /v1/admin
  • /v1/job
  • /v1/jobs
  • /v2/job
  • /v2/jobs
  • /v2/admin
  • /v2/sla
